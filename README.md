# E-Commerce Website Manual Testing Project

## Project Overview

This project demonstrates Manual Testing practices performed on an E-Commerce Web Application. The objective was to validate the functionality, usability, and reliability of key modules such as Login, Registration, Product Search, Wishlist, Shopping Cart, and Checkout.

## Testing Scope

- User Registration
- User Login
- Product Search
- Wishlist Management
- Shopping Cart
- Checkout Process

## Testing Types Performed

- Functional Testing
- Smoke Testing
- Sanity Testing
- Regression Testing
- UI Testing
- End-to-End Testing

## Tools Used

- Jira (Defect Tracking Simulation)
- Microsoft Excel
- Google Sheets
- GitHub
